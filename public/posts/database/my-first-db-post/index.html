Hugo로 첫 포스트 작성하기

Hugo를 사용하여 나의 첫 번째 블로그 포스트를 작성하는 방법을 소개합니다.

Hugo는 간단하고 빠른 정적 사이트 생성기로, 마크다운을 사용하여 콘텐츠를 작성할 수 있습니다. Hugo와 GitHub Pages를 함께 사용하면, 복잡한 서버 설정 없이도 개인 블로그를 운영할 수 있습니다.
